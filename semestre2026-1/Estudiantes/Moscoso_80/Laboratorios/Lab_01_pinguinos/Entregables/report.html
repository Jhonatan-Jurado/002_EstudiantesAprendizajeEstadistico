Paradoja de Simpson — bill_depth_mm × bill_length_mm: la correlación global es −0.235 (negativa), pero el scatter coloreado por especie (scatter_bill_depth_mm_vs_bill_length_mm_hue_species) muestra que dentro de cada especie el patrón podría diferir. 7 — Conclusiones en tres capas

A) Hallazgos descriptivos

  • 344 individuos, 0 duplicados, 0.79 % de faltantes. Dataset limpio y listo para análisis. [00_raw_profile, 03_duplicates_report]
  • Adelie es la especie dominante (44.19 %); Chinstrap la minoría (19.77 %). [04b_categorical_summary]
  • La distribución de sexos es casi perfectamente balanceada dentro de cada especie (Adelie y Chinstrap exactamente 50/50). [04c_crosstab_species_sex]
  • La asociación species×island es prácticamente determinista: Gentoo→Biscoe, Chinstrap→Dream, Adelie→las tres. [04c_crosstab_species_island]
  • flipper_length_mm es el predictor morfológico más fuerte de body_mass_g (Pearson=0.871, consistente en Spearman y Kendall). [04d_correlation_*]

B) Patrones visuales

  • Los histogramas de todas las variables numéricas muestran bimodalidad, indicando la presencia de al menos dos subpoblaciones diferenciadas.
  • Los box plots por especie muestran poco solapamiento en flipper_length_mm y body_mass_g, confirmando la separación entre grupos.
  • El scatter bill_depth_mm vs bill_length_mm coloreado por especie revela tres regiones distinguibles, sugiriendo que la correlación negativa global es un artefacto de mezcla (Paradoja de Simpson).

C) Próximas hipótesis a probar

  1. Correlaciones intra-especie: calcular Pearson/Spearman por subgrupo para confirmar la Paradoja de Simpson en bill_depth_mm×bill_length_mm.
  2. Dimorfismo sexual: Mann-Whitney U o t-test de Welch para body_mass_g entre sexos, controlando por especie.
  3. Regresión flipper→mass: ajustar un modelo lineal global y uno por especie para cuantificar la asociación y detectar interacciones.
  4. Clasificación de especie: explorar si bill_length_mm + flipper_length_mm son suficientes para separar las tres especies (inspección visual en Phase 3 lo sugiere).

8 — Preguntas para el investigador

  1. Datos faltantes en sex: ¿el sexo se registró con un protocolo sistemático o depende del estado del animal? ¿Podría la ausencia de sex correlacionar con alguna especie o isla?
  2. Control por especie: la correlación global bill_length_mm×body_mass_g (r=0.595) mezcla tres especies. ¿Se mantiene el mismo signo y magnitud dentro de cada especie?
  3. Exclusividad isla-especie: ¿la restricción geográfica observada es biológica o refleja un sesgo de muestreo? ¿Se han registrado casos de cohabitación en otras fuentes?
  4. Skewness de body_mass_g: dado que el desbalance de sexos en Gentoo no explica el skewness, ¿hay otros factores (estación, año, estado nutricional) que puedan contextualizarlo?
